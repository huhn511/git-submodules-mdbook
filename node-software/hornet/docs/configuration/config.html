config.json

Hornet uses the JSON standard as config file. If you are unsure about some syntax have a look at the official specs here. The default config file is config.json. You can change the path or name of the config file by using the -c or --config flag. For Example: hornet -c config_example.json.
You can always get the most up-to-date description of the config parameters by running hornet -h --full.
